# Logica Pro Topics

**Live Demo:** [https://logicaprotopicos.netlify.app](https://logicaprotopicos.netlify.app)

---

## What It Is

Interactive Portugol learning checklist. From basics to advanced topics.

## Features

* Checkmark topics as done
* Progress bar
* Video links (YouTube)
* Export to printable PDF
* Fully responsive

## Tech

* React + Tailwind CSS
* Lucide Icons
* Netlify hosting

## How to Use

1.  Click to check items
2.  Expand for videos
3.  Click "Generate PDF" to print

## Local Setup

```bash
npm install
npm start
